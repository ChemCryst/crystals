Making and Removing Bonds

CRYSTALS automatically creates in internal list of the bonds between atoms (LIST 41). This list is formed using the covalent radii stored for each atom type in LIST 29.
The user may wish to add to or delete bonds from LIST 41. This can be done via the graphical editor, or by adding new rules to the user-readable LIST 40
The Menu Add and Remove Bonds allows the users to edit the rules. The new rules will be applied to the current and any future atom lists (LIST 5).

If a rule refers to an atom which has been deleted from the model, a warning is issued and the remaining rules applied.
\BONDING   ACTION
           Possible Actions are:
           REPLACE (default, replaces all previous information)
           EXTEND (Adds new information to existing)
DEFAULTS   TOLTYPE= TOLERANCE= MAXBONDS= NOSYMMETRY= SIGCHANGE=
ELEMENT    TYPE= RADIUS= MAXBONDS=
PAIR       TYPE1= TYPE2= MIN= MAX= BONDTYPE=
            Possible bondtypes are:
            0 - type undefined
            1,2,3 - single, double or tripple bond
MAKE       atom-specification TO atom-specification bondtype
BREAK      atom-specification TO atom-specification
END

Example

#BONDING
BREAK FE(12) TO O(16)
MAKE O(16) TO C(17) 1
END
