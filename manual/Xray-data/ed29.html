Edit Elemental Properties

This script opens the list of elemental properties in a text editor:
# This file (ScpLst29.dat) will be read by CRYSTALS
# when you close it, so save any changes that you make.
# To abandon changes: either don't save, or, delete all
# lines from this file before saving.
# 
# THE MOST COMMON ERROR IS FORGETTING TO ENSURE
# THAT THE NUMBER OF ATOMS IS CORRECT IN THE
# THE 'READ NATOM=...' DIRECTIVE
#LIST 29
READ NELEM=  4
#             covalent        vdw    ionic     number    muA    weight colour
ELEMENT C     0.7700        1.7800   0.0100     7.000    1.150   12.011 GREE
ELEMENT H     0.3200        1.3300  -0.3000    11.000    0.062    1.008 LGRE
ELEMENT N     0.7700        1.7800  -0.1000     1.000    1.960   14.007 BLUE
ELEMENT O     0.7700        1.7800   1.3600     3.000    3.250   15.999 RED 
END

If you add new elements, dont forget to change the number of elements, NELEM= n
