Inputting Twinned Data

ROTAX

hklf4 reflection file transformation

If, during the refinement of the structure, you come to suspect that the crysal was twinned, and you are unable to reprocess the data into an hklf5 file, CRYSTALS can attempt to allocate twin component identifiers to the hklf4 data.
The utility ROTAX will analyse Fo and Fc and suggest possible twin laws. You can either accept one of these findings, or enter a twin law of your own invention.
CRYSTALS converts the untwinned reflection file to the twinned equivalent
Use the menu Analyse/ROTAX
Remember that if you have general twinning in a noncentrosymmetric space group, you may want to add (using the ROTAX interface) the corresponding inverse twin laws because post-refinement determination of the Flack parameter is unsafe.

Interpreting the Twin Law

The operator generating the twin laws can be found by computing the determinant and trace of the matrices. 

            Compute the determinant and the trace (sum  of the diagonal
            elements. Add the determinant and the trace and use Carmelo's
            little table  (GIACOVAZZO, Section 1.d)

         Tr+det:   4  0  1  2  3 -4  0 -1 -2 -3
         Element:  1  2  3  4  6 -1 -2 -3 -4 -6

e.g. for matrices:
A 1 0 0 0 1 0 0 0 1        volume fraction : 0.588
B 1 0 0 -1 -1 0 0 0 -1     volume fraction : 0.361
a -1 0 0 0 -1 0 0 0 -1     volume fraction : 0.050
b -1 0 0 1 1 0 0 0 1       volume fraction : 0

A) det 1  tr  3 total  4 element  1
B) det 1  tr -1 total  0 element  2 or -2 choose 2 because det is +ve
a) det -1 tr -3 total -4 element -1
b) det -1 tr  1 total  0 element  2 or -2  choose -2 (i.e. m) because det is -ve

hklf5 reflection file input

ROTAX is not the appropriate utility for inputting hklf5 data

If you discovered that your crystal was twinned while it was on the diffractometer, you should have created a reflection file in hklf5 format.
This format is incompatible with CRYSTALS, but the utility hklf52cry will convert the format. It will also attempt to analyse the reflections and derive the Twin laws
Use the menu Data/import hklf5 file or
Use Data/Open File and select the hklf5 file. Dont forget to tick the box SHELX hklf5 file if this has not been automatically selected
