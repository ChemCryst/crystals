Installation
############
.. index:: Installing CRYSTALS


Download the latest CRYSTALS installer from http://www.xtl.ox.ac.uk/.

1. Double click the downloaded file to start the installation.
2. Choose a location to install, e.g. C:\Users\me\AppData\Local\Programs\crystals
